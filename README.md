# Paint

Một ứng dụng vẽ đơn giản được xây dựng nhằm cung cấp các công cụ sáng tạo cơ bản trực tiếp trên máy tính

## Tính năng chính

* **Công cụ vẽ:** Bút chì, tẩy, vẽ hình khối (vuông, tròn, đường thẳng)
* **Bảng màu:** Tùy chọn màu sắc đa dạng cho nét vẽ và nền
* **Tùy chỉnh:** Điều chỉnh độ dày mỏng của nét vẽ (Stroke size)
* **Tiện ích:** Xóa toàn bộ canvas, gom nhóm các hình với nhau
